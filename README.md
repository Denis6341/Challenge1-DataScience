# Análisis de Ventas de Tiendas

## Descripción del proyecto

Este proyecto realiza un análisis exploratorio de datos sobre las ventas de cuatro tiendas diferentes. El objetivo es evaluar el desempeño de cada tienda para determinar cuál representa la mejor opción comercial para que el Sr. Juan venda sus productos.

El análisis se realizó utilizando Python en Google Colab y empleando bibliotecas de ciencia de datos como Pandas, Matplotlib y Seaborn.

---

## Objetivos del análisis

* Calcular los ingresos totales de cada tienda
* Analizar las categorías de productos más vendidas
* Evaluar la satisfacción de los clientes mediante las calificaciones
* Identificar los productos más y menos vendidos
* Analizar el costo promedio de envío
* Explorar patrones geográficos en las ventas

---

---

## Resultados

El análisis permitió identificar las diferencias de desempeño entre las tiendas en términos de ingresos, satisfacción del cliente y comportamiento de ventas.

Los resultados ayudan a tomar decisiones estratégicas basadas en datos.
